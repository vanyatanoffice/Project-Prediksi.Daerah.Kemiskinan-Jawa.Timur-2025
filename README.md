# Project-Prediksi.Daerah.Kemiskinan-Jawa.Timur-2025
Dashboard ini merupakan aplikasi berbasis Streamlit untuk menganalisis tingkat kemiskinan kabupaten/kota di Jawa Timur. Fitur yang tersedia yaitu data overview, analisis korelasi, peta cluster, dan prediksi daerah menggunakan KNN berdasarkan 7 indikator ekonomi, seperti Garis Kemiskinan, TPT, P1, P2, Gini Rasio, dan Rasio Kesejahteraan.
